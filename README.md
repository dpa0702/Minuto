# Minuto
Teste Minuto Seguros


Criar um programa para obter automaticamente o conteúdo dos dez últimos tópicos publicados no blog da Minuto Seguros. Abaixo segue um link para auxiliá-lo nesse trabalho: 
 
http://www.minutoseguros.com.br/blog/feed/ 


Ao executar o projeto carregará o Swagger.

Para testar, submeter o método GET
/Minutos/GetMinutoPostsAndCreate
(Try it out --> Execute)

Caso tenha resposta Code 200 ou Status 200 tudo ocorreu bem e pode passar ao próximo passo.

Submeter ao método GET
/Minutos
(Try it out --> Execute) *** Não é necessário informar parâmetros.
Analisar o retorno da leitura dos artigos e análise do textos.
As principais palavras são encontradas na lista pPalavras.
O número de ocorrências das principais palavras está em ocorrencias.
A quantidade de palavras por tópicos está em quantidade.
